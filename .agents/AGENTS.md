# Agent Rules for Pawan Chaudhary Website

## Automatic Commits
Whenever you make changes to the codebase and verify that they are working, you MUST automatically commit and push those changes to the remote GitHub repository (`main` branch) without asking the user for permission.
You should do this at the end of every successful task or major milestone.
Do not ask the user if they want to commit or push; just do it automatically.
